Dashboard Siswa — Demo (PHP + JSON backend)

Cara menjalankan (lokal):
1. Extract folder ke dalam folder web server (contoh: /var/www/html/dashboard_siswa_project)
2. Pastikan PHP sudah terinstal (minimal PHP 7+).
3. Akses http://localhost/dashboard_siswa_project/index.html
4. Login: username=ppkn password=123 (demo)
5. Pada dashboard, ubah status kehadiran dan tekan Submit — file attendance_YYYY-MM-DD.json akan tersimpan di folder data/.

Catatan:
- Ini adalah demo sederhana tanpa autentikasi server-side.
- Untuk produksi, gunakan database (MySQL) dan server-side login.
